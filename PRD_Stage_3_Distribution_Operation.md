# PRD Stage 3: 다채널 배포 및 리드 웜업 시스템

## 1. 목적
제작된 콘텐츠를 자동으로 업로드하고, 고객의 반응에 실시간으로 대응하여 계정 성장을 가속화함.

## 2. 주요 기능 및 사용자 시나리오
### F1. 스마트 스케줄링 배포
*   **동작**: n8n 워크플로우가 대기 중인 콘텐츠를 플랫폼별 '골든 타임'에 맞춰 순차 발행.
*   **보안**: 계정 차단 방지를 위해 IP 순환(Proxy) 및 인간다운 지연 시간(Random Wait) 적용.

### F2. 하이브리드 업로드 (API + Playwright)
*   **동작**: 
    *   **API 지원 채널**: 즉시 HTTP Request를 통해 발행.
    *   **비지원 채널**: n8n이 Playwright 서버로 명령을 보내 실제 브라우저에서 로그인 및 포스팅 수행.

### F3. 리드 웜업 및 자동 대응 (Engagement)
*   **동작**: 
    *   게시물에 댓글이 달리면 n8n이 감지.
    *   AI가 댓글 내용을 분석하여 감사 인사 또는 질문에 대한 답변 작성.
    *   DM 요청 시 미리 설정된 마케팅 자동화 툴(Linktree/오픈채팅)로 연결.

## 3. 기술 스택
*   **Automation**: n8n (Wait, Trigger, Merge 노드 활용)
*   **UI Automation**: Playwright + Chromium (Headless 모드)
*   **Communication**: Slack API (발행 결과 및 댓글 알림)

## 4. 성공 지표 (KPI)
*   월간 발행 성공률 98% 이상.
*   댓글 반응 속도 (평균 30분 이내 자동 응대).
*   자동화를 통한 신규 팔로워 및 리드(DB) 확보 수치 측정.
