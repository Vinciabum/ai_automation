# FastCampus SNS Automation Course Source Content (Scraped)

이 문서는 Playwright 기반 추출 도구를 사용하여 `https://fastcampus.co.kr/biz_online_aisns` 페이지에서 수집한 상세 커리큘럼 및 기술 스택 정보입니다.

## 1. 핵심 가치 및 목표
*   **"잠드는 사이, 1일 3콘텐츠!"**: 클릭 한 번으로 콘텐츠 기획부터 제작, 다채널 업로드까지 전 과정 자동화.
*   **트렌드 추적**: 구독 채널 정보를 자동 수집하고 급상승 콘텐츠를 파악하는 대시보드 구축.
*   **효율성**: 고성과 영상 구조와 데이터 대시보드를 기반으로 10가지 주제 자동 생성.
*   **다채널 전략**: 계정 제한을 방지하면서 인스타그램, 유튜브 등 여러 SNS 플랫폼에 동시 배포.

## 2. 주요 기술 스택 (Tech Stack)
*   **메인 엔진**: **n8n** (워크플로우 자동화), **Claude Code** (AI 기반 코딩 및 자동화 보조).
*   **AI 모델**: Gemini (Google AI Studio), ChatGPT, Perplexity.
*   **콘텐츠 제작**: TTS (Text-to-Speech) 오디오 생성, AI 기반 영상/이미지 생성 도구.
*   **인프라**: API 연동, Webhook, 클라우드 자동화 환경.

## 3. 상세 커리큘럼 (10개 파트, 총 101개 클립)
*   **Part 1: 콘텐츠 시스템의 본질 (6개 클립)**: 지속 가능한 SNS 시스템 구축 기초.
*   **Part 2: 자동화 툴의 기초 (14개 클립)**: n8n, Claude Code 설정 및 기본 API 연결.
*   **Part 3: 기획 자동화 - 마켓 리딩 대시보드 (11개 클립)**: 트렌드 및 바이럴 콘텐츠 추적 대시보드 구축.
*   **Part 4: 콘텐츠 제작 자동화 (20개 클립)**: 대본, 이미지, 영상 제작 전 과정 자동화.
*   **Part 5: 리퍼퍼징 시스템 (14개 클립)**: 하나의 콘텐츠를 다양한 포맷으로 변환.
*   **Part 6: 멀티 채널 배포 자동화 I (2개 클립)**: 크로스 플랫폼 포스팅 초기 설정.
*   **Part 7: 자동화 툴의 기초 (심화) (6개 클립)**: 고급 자동화 로직 및 에러 핸들링.
*   **Part 8: 멀티 채널 배포 자동화 II (12개 클립)**: 전 플랫폼 배포 확장 및 스케줄링.
*   **Part 9: 운영 자동화 - 리드 웜업 시스템 (8개 클립)**: 인게이지먼트 및 리드 생성 자동화.
*   **Part 10: 시스템 완성 및 확장 (8개 클립)**: 워크플로우 최종 점검 및 다중 계정 확장.

## 4. 운영 비용 및 도구
*   **필수**: Claude Code Pro ($20/월), n8n (일부 유료/무료 티어).
*   **권장**: Gemini/ChatGPT/Perplexity (API 사용료 약 $5 내외), TTS ($5-10).
